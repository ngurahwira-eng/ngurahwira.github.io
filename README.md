# Weather Dashboard

Simple responsive Weather Dashboard using OpenWeatherMap.

Setup
- Get an API key from https://openweathermap.org/ (free tier).
- Open `script.js` and replace the `API_KEY` value with your key.

Run
- Open [index.html](index.html) directly in a browser, or run a local server for best results:

  ```bash
  python -m http.server 8000
  ```

Notes
- This uses the OpenWeatherMap Current Weather API.
- If you want, I can switch this to a different provider that doesn't require a key.
